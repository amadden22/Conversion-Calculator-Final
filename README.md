# Final

Description: Final Project Calculator for iOS Development Class. This app makes use of a previous project where the visual interface was designed. Now, the final version of the app has the ability to convert numbers from farentheit to celcius, celcius to farenheit, miles to kilometers, and kilometers to miles. All buttons and functions (clear, plus/minus, convert) are fully functional and work cleanly. 

Purpose: Create a fully functional conversion calculator iOS app. This app makes us of alerts, action sheets, case statements, IBactions for multiple interface buttons, image assets, and many other swift capabilities.
